BACKGROUND

Property Value Keterangan
Background color color apapun mengatur warna pada background
Background Image url(../image.png) mengatur gambar yang akan digunakan pada background
Background-repeat repeat mengatur jenis pengulangan gambar pada background
repeat-y gambar diulang dari sumbu y (dari atas kebawah)
repeat-x gambar diulang dari sumbu x (dari kiri ke kanan)
no-repeat gambar tidak diulang,hanya memunculkan satu gambar
Background-position posisinya mengatur posisi pada gambar background
top left berada pada atas kiri
center berada ditengah tengah
bottom right berada dibawah kanan
center left berada ditengah kiri
x% y% kirinya berapa persen dan kanan nya berapa
100px 200px kiri kekanan nya 100px dan atas kebawah nya 200px

PSEUDO CLASS

-------------
properti 
value
Keterangan
Pseudo class type Pseudo class adalah kelas semu yang dimiliki semua element HTML,yang membuat kita dapat mendefinisikan style pada keadaan tertentu dari element tersebut
:link mengubah style default pada sebuah link contoh seperti a yang memiliki href
:active style ketika sebuah link diklick(keadaan aktif)
:visited style ketika sebuah link sudah pernah dikunjungi sebelumnya
:first-child memilih element pertama dari sebuah parent
:last-child memilih elent terakhir dari sebuah parent
:nth-child(n) memilih element sesuai yang kita mau dari sebuah parent
n bisa berarti berurutan (1) (2),atau pola (2n),(3n+2),(4n-1) atau ganjil genap (even & (odd))
:target untuk membuat apa yang ingin terjadi setelahnya

<track> untuk membuat trek suara untuk elemen media

LINKS

<link> tag untuk menyambungkan antara document dan sumber daya eksternal,paling sering digunakan untuk link css
<nav> tag untuk membuat navigasi

TABLES
<table> untuk membuat table
<caption> untuk membuat sebuah caption table
<th> tag untuk membuat header table
<tr> untuk membuat baris dalam sebuah table
<td> untuk membuat sel dalam sebuah table/isi dari table
<thead> untuk mengelompokan isi header dalam sebuah table
<tbody> untuk mengelompokan isi body dalam sebuah table
<tfoot> untuk mengelompokan isi footer dalam sebuah table
<col> menentukan properti kolom untuk setiap kolom dalam element colgroup
<colgroup> menentukan kelompok dari satu atau lebih kolom dalam sebuah table untuk diformat

STYLE/SECTION
<style> untuk membuat styling pada html
<div> untuk mengelompokan bagian bagian html
<span> untuk mengelompokan kata kata
<header> untuk mengelompokan bagian header html
<footer> untuk mengelompokan bagian bawah/footer document
<hgroup> pengelompokan elemen heading
<section> untuk mengelompokan bagian pada body
<article> untuk membuat sebuah article
<aside> bertujuan untuk menandai bagian web yang bukan berisi konten utama, tetapi memiliki kaitan dengan artikel yang saat ini ditampilkan. Bagian paling pas untuk tag aside adalah sidebar
<details> membuat rincian tambahan yang pengguna dapat lihat atau sembunyikan
<dialog> untuk membuat sebuah kotak dialog atau jendela
<summary> Tag untuk membuat sebuah judul terlihat untuk elemen

Meta Info
<head> untuk membuat informasi tentang dokumen
<meta> untuk membuat metadata tentang dokumen HTML
<base> Menentukan URL dasar / target untuk semua URL relatif dalam dokumen

SCRIPT
<script> Tag untuk membuat script di sisi klien
<noscript> untuk membuat sebuah konten alternatif bagi pengguna yang tidak mendukung script di sisi klien
<embed> Tag untuk membuat sebuah wadah untuk aplikasi eksternal (non-HTML) (tag baru HTML5)
<object> Tag untuk membuat sebuah objek yang ditanam
<param> Tag untuk membuat sebuah parameter untuk objek
